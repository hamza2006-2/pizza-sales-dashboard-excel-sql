# 📊 Dashboard

## Overview
This folder contains the Microsoft Excel Dashboard developed for the Pizza Sales Analysis project.

## Features
- Interactive Dashboard
- KPI Cards
- Pivot Tables
- Pivot Charts
- Slicers
- Business Insights

## Dashboard Includes
- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Category Analysis
- Sales Trend
- Top Selling Pizzas
- Bottom Selling Pizzas

## Purpose
To present business insights in a simple, interactive, and visually appealing format.
